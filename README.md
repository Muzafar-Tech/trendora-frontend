# 🎨 Trendora Frontend

A modern frontend for **Trendora**, a premium SaaS Creator Marketplace connecting brands with content creators through campaigns, collaborations, and opportunities.

Built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**, the application delivers a premium dashboard experience inspired by Stripe, Linear, Vercel, and Notion while maintaining a fully responsive and scalable architecture.

## ✨ Features

- Modern SaaS UI
- Responsive Design
- Role-Based Dashboards
- Creator Dashboard
- Brand Dashboard
- Admin Dashboard
- Authentication System
- Email Verification
- Password Reset
- Premium Animations
- Protected Routes
- Reusable Components
- Nested Routing Architecture
- Iconify Icons
- API Integration
- Mobile Friendly

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Axios
- Iconify
